Swift Craft Launcher News API

Welcome to the Swift Craft Launcher News API documentation.

API Endpoints

Get Announcement
GET /api/announcements/{version}/{lang}.json

Parameters

  • version - Version number (e.g., 0.3.1-beta)
  • lang - Language code (e.g., en, zh-Hans)

Example

https://suhang12332.github.io/Swift-Craft-Launcher-News/api/announcements/0.3.1-beta/en.json

Supported Languages

ar, da, de, en, es, fi, fr, hi, it, ja, ko, nb, nl, pl, pt, ru, sv, th, tr, vi, zh-Hans, zh-Hant
